### Form Validation
A basic login form with validations!

### Use of the Project:
We can add this login form at start of any website to verify deatils.

### Tech Stack
* HTML5
* CSS3
* JAVASCRIPT
* REGEX

#### Steps to Use:

---

- Download or clone the repository


- Go to the directory
- Run the index.html file
- Start validating!!!
